## Installation rapide
1. Extraire l'archive : tar -xzf burkina-watch-portable.tar.gz
2. Installer les dépendances : ./setup.sh
3. Configurer .env avec vos variables
4. Lancer l'application : ./start.sh

## Prérequis
- Node.js 18+
- PostgreSQL 14+
- npm 9+

Pour plus de détails, consultez README_PORTABLE.md
